Eric Meyer Reset Template

Summary

This resets all browser based default styles to a base level negated state. All browsers (or, more precisely, browser engine families) have default styles for the displayable HTML elements. Because the HTML standard(s) did not specify default styles, browser engine implementers chose the default settings arbitrarily. The result, unfortunately, is that many of the styles differ from browser engine to browser engine. The idea behind a reset style sheet is to normalize the styles for all displayable HTML elements to a base state that eliminates all browser engine based style differences.

Notes:

  • Eric Meyer wasn't the first person to promote a reset stylesheet, that was Tan; but, he was the one to promote and popularize the notion.
  • His current reset style sheet is the one used in this HTML page.
  • Eric Meyer promoted the idea of resetting as one of stripping all formatting from the major elements. It was from this stripped state that document or site-specific styles would be added back.
  • As of 3/4/15, you will note that the reset stylesheet does not remove the yellow from the mark inline element. It also does not remove the centering of the table caption. As such, not all formatting is stripped, and his reset stylesheet can be considered a work in progress.
  • You can find a useful history of css resets here
  • You can also find useful information on all things css reset at the css reset site
